# Kisan Kamai Production App

This app is the Firebase-only production build for `www.kisankamai.com`.

## Local development

```bash
cd apps/production
npm install
npm run dev
```

## Firebase App Hosting

- Reuse Firebase project: `gokisaan`
- App root directory: `apps/production`
- Canonical domain: `www.kisankamai.com`
- Apex redirect source: `kisankamai.com`

## Required production services

- Firebase App Hosting
- Firebase Authentication
- Cloud Firestore
- Cloud Storage
- Optional Sentry DSN for error monitoring

## GoDaddy

In GoDaddy DNS, add only the records generated by Firebase App Hosting for:

- `www.kisankamai.com`
- `kisankamai.com`

Remove old forwarding or conflicting `A`, `AAAA`, or `CNAME` records for `@` and `www`.
