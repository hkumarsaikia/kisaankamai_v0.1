# Runtime Logs

This folder is the canonical location for local runtime and debug log captures.

Use it for files such as:

- `dev-runtime-3000.log`
- `start-runtime-3207.log`
- `dev-server.out.log`
- `dev-auth.err.log`

Framework entrypoints like `instrumentation.ts` stay at the repo root and are not part of this folder.
